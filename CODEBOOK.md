Getting-and-Cleaning-Data-Course-Project
==========================================

### Modifications

* Merging datasets
* Extracting measurements on the mean and standard deviation for each measurement
* Replacing activity numbers with the descriptive activity name
* Replacing dataset variable names with the descriptive variable name
* Creating a final tidy dataset using all the changes listed above


## Descriptions

### Identififiers
* Subject: The Subject's ID
* Activity: The Activity name that measurements were recorded off of
